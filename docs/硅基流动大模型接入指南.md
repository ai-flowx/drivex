# 硅基流动接入指南

硅基流动文档地址：[https://docs.siliconflow.cn/cn/userguide](https://docs.siliconflow.cn/cn/userguide)

**接入步骤**
1. 用户前往硅基流动的[SiliconCloud平台](https://cloud.siliconflow.cn/)创建API密钥
3. 获取到api_key之后，就可以开始填入到config.json

`simple-one-api`对应的配置文件示例：
```json
{
  "server_port": ":9090",
  "load_balancing": "random",
  "services": {
    "siliconflow": [
      {
        "models": ["deepseek-ai/DeepSeek-R1-Distill-Llama-8B"],
        "enabled": true,
        "credentials": {
          "api_key": "key"
        },
        "server_url":"https://api.siliconflow.cn/v1"
      }
    ]
  }
}

```
**设置模型别名**
如果需要设置对外的模型名称，可以设置参数`model_map`，从而起到别名作用；
例如这里`deepseek-ai/DeepSeek-R1-Distill-Llama-8B`别名为`deepseek8b`，从而在调用`simple-one-api`的时候传入`deepseek8b`即可；
`simple-one-api`对应的配置文件示例：
```

```json
{
  "server_port": ":9099",
  "load_balancing": "random",
  "services": {
    "siliconflow": [
      {
        "models": ["deepseek8b"],
        "enabled": true,
        "credentials": {
          "api_key": "key"
        },
        "model_map":{
          "deepseek8b": "deepseek-ai/DeepSeek-R1-Distill-Llama-8B"
        },
        "server_url":"https://api.siliconflow.cn/v1"
      }
    ]
  }
}
```
